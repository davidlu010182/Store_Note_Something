C++ bind function
簡而言之，就是對已有的一個function進行包裝(wrap)，固定其中的幾個參數(argument)。
如
int add(int a, int b){return a+b;} //這是一個未包裝過的function，有兩個argument。
auto add12 = bind(add, 1, 2); //包裝add，固定兩個參數。
auto add2 = bind(add, placeholders::_1, 2); //只固定第二個參數，放開對第一個參數的控制。
int ans = add2(5); //相等於add(5, 2)，5被代入placeholders_::1，

placeholders(佔位符)的順序可以隨意打亂，只要能確定對應的參數
auto add88 = bind(add, placeholders::_2, placeholders::_1);
int ans = add88(8, 5); //相等於add(5, 8)

類別的函式呼叫
Time.show(55, 12); //這是一般使用時。
可實際上，compile時會是:
show(&Time, 55, 12); //傳入呼叫該函式的類別的位址

因此，在wrap類別內的成員函式時，需要加上一個表示呼叫者的指標(若需要用到該類成員)，除非是static成員。
且該類指標屬於第一個參數，佔位符也是 placeholders::_1。 
如
auto Time55 = bind(Time::show, placeholders::_1, 55, placeholders:_3);
若是在Time類中包裹，則該佔位符可直接改成this，代表在該類中呼叫時，傳入自身位址。
